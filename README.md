# Copilot Skills

A collection of reusable Copilot CLI skills for engineering workflows in Dynamics 365 / Service Intelligence projects.

## Skills

| Skill | Description |
|---|---|
| `CCA-co-dev-spec-and-build` | End-to-end CCA feature development workflow: PM ask → codebase review → tech plan → tech spec → code change |

## How to Use

Each skill is an instruction file that guides Copilot CLI through a structured, repeatable workflow. Reference the skill at the start of a session or drop the instruction file into your repo's `.github/instructions/` folder.
